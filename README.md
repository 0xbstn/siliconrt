Experimental Apple Silicon inference substrate.

`siliconrt` is a low-level lab for Apple-native inference building blocks:
C++ host runtime paths, Metal-backed storage, and small kernel-oriented
experiments for Mac.

It is intentionally narrow, unstable, and early.

## Scope

- low-level runtime and memory experiments
- Metal-backed storage and buffer paths
- small native microbenches and operator prototypes

## Status

Alpha.

Interfaces can change aggressively and current code should be treated as
experimental research infrastructure, not a stable library.
